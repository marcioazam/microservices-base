# Optics

Functional optics for immutable data manipulation.

## Packages

| Package | Description |
|---------|-------------|
| `lens` | Lens for focusing on nested data |
| `prism` | Prism for optional data access |

## Usage

```go
import "github.com/auth-platform/libs/go/optics/lens"
import "github.com/auth-platform/libs/go/optics/prism"
```
