# Utils

General utilities and helpers.

## Packages

| Package | Description |
|---------|-------------|
| `audit` | Audit logging |
| `cache` | Caching utilities |
| `codec` | Encoding/decoding (JSON, YAML) |
| `diff` | Diff utilities |
| `error` | Error response handling |
| `merge` | Deep merge utilities |
| `uuid` | UUID generation |
| `validated` | Validated types |
| `validator` | Validation utilities |

## Usage

```go
import "github.com/auth-platform/libs/go/utils/uuid"
import "github.com/auth-platform/libs/go/utils/codec"
```
