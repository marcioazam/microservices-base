# Concurrency

Concurrency primitives and utilities for Go.

## Packages

| Package | Description |
|---------|-------------|
| `async` | Async/await patterns for Go |
| `atomic` | Atomic operations wrapper |
| `channels` | Channel utilities |
| `errgroup` | Error group for concurrent operations |
| `once` | Enhanced once execution |
| `pool` | Worker pool implementation |
| `syncmap` | Thread-safe map wrapper |
| `waitgroup` | Enhanced wait group |

## Usage

```go
import "github.com/auth-platform/libs/go/concurrency/async"
import "github.com/auth-platform/libs/go/concurrency/pool"
```
