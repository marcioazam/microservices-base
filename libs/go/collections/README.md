# Collections

Data structures and collection utilities for Go.

## Packages

| Package | Description |
|---------|-------------|
| `lru` | Least Recently Used cache implementation |
| `maps` | Map utilities and operations |
| `pqueue` | Priority queue implementation |
| `queue` | FIFO queue implementation |
| `set` | Set data structure with common operations |
| `slices` | Slice utilities and functional operations |
| `sort` | Sorting utilities |

## Usage

```go
import "github.com/auth-platform/libs/go/collections/slices"
import "github.com/auth-platform/libs/go/collections/set"
```
