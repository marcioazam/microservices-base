# Server

Server utilities and infrastructure.

## Packages

| Package | Description |
|---------|-------------|
| `health` | Health check endpoints |
| `shutdown` | Graceful shutdown handling |
| `tracing` | Distributed tracing utilities |

## Usage

```go
import "github.com/auth-platform/libs/go/server/health"
import "github.com/auth-platform/libs/go/server/shutdown"
```
