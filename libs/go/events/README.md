# Events

Event handling and messaging utilities.

## Packages

| Package | Description |
|---------|-------------|
| `builder` | Event builder pattern |
| `eventbus` | In-memory event bus |
| `pubsub` | Publish/subscribe pattern |

## Usage

```go
import "github.com/auth-platform/libs/go/events/eventbus"
import "github.com/auth-platform/libs/go/events/pubsub"
```
